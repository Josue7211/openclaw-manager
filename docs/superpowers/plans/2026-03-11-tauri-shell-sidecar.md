# Tauri Shell + Sidecar — Implementation Plan (Phase 1 of 3)

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Get mission-control running as a Tauri v2 desktop app with the existing Next.js server as a Node.js sidecar — no database changes yet, Supabase still works.

**Architecture:** Tauri wraps a webview pointing to `localhost:3000`. Rust spawns a Node.js sidecar process that runs the Next.js standalone server. Secrets are read from OS keychain and injected as env vars. The app works identically to the current web version but in a native window.

**Tech Stack:** Tauri v2, Rust, `tauri-plugin-shell` (sidecar), `tauri-plugin-notification`, `tauri-plugin-store` (keychain), Next.js `output: 'standalone'`

**Spec:** `docs/superpowers/specs/2026-03-11-tauri-desktop-app-design.md`

### Task 3: Create the sidecar bundle script

**Files:**
- Create: `scripts/bundle-sidecar.sh`

The sidecar for Tauri must be a single executable named `node-<target-triple>`. We bundle the platform's Node.js binary into `src-tauri/binaries/`.

- [ ] **Step 1: Create `scripts/bundle-sidecar.sh`**

```bash
#!/usr/bin/env bash
set -euo pipefail

# Detect current platform's target triple
case "$(uname -s)-$(uname -m)" in
  Linux-x86_64)   TARGET="x86_64-unknown-linux-gnu" ; NODE_PLATFORM="linux-x64" ;;
  Linux-aarch64)  TARGET="aarch64-unknown-linux-gnu" ; NODE_PLATFORM="linux-arm64" ;;
  Darwin-x86_64)  TARGET="x86_64-apple-darwin"       ; NODE_PLATFORM="darwin-x64" ;;
  Darwin-arm64)   TARGET="aarch64-apple-darwin"       ; NODE_PLATFORM="darwin-arm64" ;;
  MINGW*|MSYS*)   TARGET="x86_64-pc-windows-msvc"    ; NODE_PLATFORM="win-x64" ;;
  *)              echo "Unsupported platform"; exit 1 ;;
esac

NODE_VERSION="22.16.0"
BINARIES_DIR="src-tauri/binaries"
mkdir -p "$BINARIES_DIR"

echo "==> Downloading Node.js $NODE_VERSION for $NODE_PLATFORM..."

if [[ "$NODE_PLATFORM" == win-* ]]; then
  NODE_URL="https://nodejs.org/dist/v${NODE_VERSION}/node-v${NODE_VERSION}-${NODE_PLATFORM}.zip"
  curl -fsSL "$NODE_URL" -o /tmp/node.zip
  unzip -qo /tmp/node.zip -d /tmp/node-extract
  cp "/tmp/node-extract/node-v${NODE_VERSION}-${NODE_PLATFORM}/node.exe" "${BINARIES_DIR}/node-${TARGET}.exe"
  rm -rf /tmp/node.zip /tmp/node-extract
else
  NODE_URL="https://nodejs.org/dist/v${NODE_VERSION}/node-v${NODE_VERSION}-${NODE_PLATFORM}.tar.xz"
  curl -fsSL "$NODE_URL" -o /tmp/node.tar.xz
  tar -xf /tmp/node.tar.xz -C /tmp
  cp "/tmp/node-v${NODE_VERSION}-${NODE_PLATFORM}/bin/node" "${BINARIES_DIR}/node-${TARGET}"
  chmod +x "${BINARIES_DIR}/node-${TARGET}"
  rm -rf /tmp/node.tar.xz "/tmp/node-v${NODE_VERSION}-${NODE_PLATFORM}"
fi

echo "==> Node.js binary ready at ${BINARIES_DIR}/node-${TARGET}"

echo "==> Building Next.js standalone..."
npm run build

echo "==> Copying static assets into standalone..."
cp -r .next/static .next/standalone/.next/static
cp -r public .next/standalone/public 2>/dev/null || true

echo "==> Sidecar bundle ready."
```

```rust
/// Read all secrets from the store and return as env var map
pub fn load_env_vars(app: &AppHandle) -> HashMap<String, String> {
    let store = app.store(STORE_FILE).expect("failed to open secret store");
    let mut env = HashMap::new();

    for (key, env_name) in KEY_ENV_MAP {
        if let Some(val) = store.get(*key) {
            if let Some(s) = val.as_str() {
                if !s.is_empty() {
                    env.insert(env_name.to_string(), s.to_string());
                }
            }
        }
    }

    // Auto-generate MC_API_KEY if missing
    if !env.contains_key("MC_API_KEY") {
        let key = generate_api_key();
        store.set(
            "mc-api-key".to_string(),
            serde_json::Value::String(key.clone()),
        );
        store.save().ok();
        env.insert("MC_API_KEY".to_string(), key);
    }

    env
}

/// Check if this is the first run (no secrets configured)
pub fn is_first_run(app: &AppHandle) -> bool {
    let store = app.store(STORE_FILE).expect("failed to open secret store");
    // If only mc-api-key exists (auto-generated), it's still first run
    let has_user_config = KEY_ENV_MAP.iter().any(|(key, _)| {
        *key != "mc-api-key"
            && store
                .get(*key)
                .and_then(|v| v.as_str().map(|s| !s.is_empty()))
                .unwrap_or(false)
    });
    !has_user_config
}

/// Generate a random 256-bit hex API key
fn generate_api_key() -> String {
    use rand::Rng;
    let mut rng = rand::thread_rng();
    let bytes: [u8; 32] = rng.gen();
    hex::encode(bytes)
}

/// Tauri command: get a secret value
#[tauri::command]
pub fn get_secret(app: AppHandle, key: String) -> Option<String> {
    let store = app.store(STORE_FILE).expect("failed to open secret store");
    store.get(&key).and_then(|v| v.as_str().map(String::from))
}

/// Tauri command: set a secret value
#[tauri::command]
pub fn set_secret(app: AppHandle, key: String, value: String) {
    let store = app.store(STORE_FILE).expect("failed to open secret store");
    store.set(key, serde_json::Value::String(value));
    store.save().ok();
}

/// Tauri command: get all configured module statuses
#[tauri::command]
pub fn get_modules(app: AppHandle) -> HashMap<String, bool> {
    let store = app.store(STORE_FILE).expect("failed to open secret store");
    let has = |key: &str| -> bool {
        store
            .get(key)
            .and_then(|v| v.as_str().map(|s| !s.is_empty()))
            .unwrap_or(false)
    };

    let mut modules = HashMap::new();
    modules.insert("messages".into(), has("bluebubbles.host"));
    modules.insert("calendar".into(), has("caldav.url"));
    modules.insert("homelab".into(), has("proxmox.host") || has("opnsense.host"));
    modules.insert("media".into(), has("plex.url") || has("sonarr.url") || has("radarr.url"));
    modules.insert("email".into(), has("email.host"));
    modules.insert("chat".into(), has("openclaw.ws"));
    modules.insert("agents".into(), has("anthropic.api-key"));
    modules
}

/// Tauri command: check if first run
#[tauri::command]
pub fn check_first_run(app: AppHandle) -> bool {
    is_first_run(&app)
}
```

- [ ] **Step 1: Create `src-tauri/src/sidecar.rs`**

```rust
use std::collections::HashMap;
use std::time::Duration;
use tauri::AppHandle;
use tauri_plugin_shell::ShellExt;
use tauri_plugin_shell::process::{CommandChild, CommandEvent};

/// Spawn the Node.js sidecar running the Next.js standalone server.
/// Returns the child process handle for later cleanup.
pub async fn spawn_sidecar(
    app: &AppHandle,
    env_vars: HashMap<String, String>,
) -> Result<CommandChild, String> {
    let mut cmd = app
        .shell()
        .sidecar("binaries/node")
        .map_err(|e| format!("Failed to create sidecar command: {e}"))?;

    // Pass the standalone server.js path as argument
    // The sidecar binary is Node.js — we tell it to run server.js
    // In production, server.js is bundled alongside the app
    let server_path = get_server_js_path(app);

    let mut cmd = app
        .shell()
        .sidecar("binaries/node")
        .map_err(|e| format!("Failed to create sidecar command: {e}"))?
        .args([&server_path])
        .envs(env_vars)
        .env("PORT", "3000")
        .env("HOSTNAME", "127.0.0.1")
        .env("NODE_ENV", "production");

    let (mut rx, child) = cmd
        .spawn()
        .map_err(|e| format!("Failed to spawn sidecar: {e}"))?;

    // Log sidecar stdout/stderr in background
    tauri::async_runtime::spawn(async move {
        while let Some(event) = rx.recv().await {
            match event {
                CommandEvent::Stdout(line) => {
                    let text = String::from_utf8_lossy(&line);
                    println!("[sidecar] {}", text.trim());
                }
                CommandEvent::Stderr(line) => {
                    let text = String::from_utf8_lossy(&line);
                    eprintln!("[sidecar:err] {}", text.trim());
                }
                CommandEvent::Terminated(status) => {
                    println!("[sidecar] terminated with status: {:?}", status);
                    break;
                }
                _ => {}
            }
        }
    });

    Ok(child)
}

/// Wait for the sidecar to be ready by polling the health endpoint
pub async fn wait_for_ready(timeout_secs: u64) -> Result<(), String> {
    let client = reqwest::Client::new();
    let deadline = tokio::time::Instant::now() + Duration::from_secs(timeout_secs);

    loop {
        if tokio::time::Instant::now() > deadline {
            return Err("Sidecar failed to start within timeout".to_string());
        }

        match client.get("http://127.0.0.1:3000/api/health").send().await {
            Ok(resp) if resp.status().is_success() => {
                println!("[sidecar] ready!");
                return Ok(());
            }
            _ => {
                tokio::time::sleep(Duration::from_millis(500)).await;
            }
        }
    }
}

/// Resolve the path to .next/standalone/server.js
fn get_server_js_path(app: &AppHandle) -> String {
    // In development, it's relative to the project root
    // In production, it's bundled with the app resources
    let resource_dir = app
        .path()
        .resource_dir()
        .unwrap_or_default();

    let prod_path = resource_dir.join("standalone/server.js");
    if prod_path.exists() {
        return prod_path.to_string_lossy().to_string();
    }

    // Fallback for development
    ".next/standalone/server.js".to_string()
}
```

```rust
#![cfg_attr(not(debug_assertions), windows_subsystem = "windows")]

mod secrets;
mod sidecar;

use std::sync::Mutex;
use tauri::Manager;
use tauri_plugin_shell::process::CommandChild;

struct SidecarState(Mutex<Option<CommandChild>>);

fn main() {
    tauri::Builder::default()
        .plugin(tauri_plugin_shell::init())
        .plugin(tauri_plugin_notification::init())
        .plugin(tauri_plugin_store::Builder::default().build())
        .manage(SidecarState(Mutex::new(None)))
        .invoke_handler(tauri::generate_handler![
            secrets::get_secret,
            secrets::set_secret,
            secrets::get_modules,
            secrets::check_first_run,
        ])
        .setup(|app| {
            let app_handle = app.handle().clone();

            // Load env vars from secret store
            let env_vars = secrets::load_env_vars(&app_handle);
            let first_run = secrets::is_first_run(&app_handle);

            if first_run {
                // In first run, skip sidecar — webview will show setup wizard
                // Sidecar gets started after setup completes
                println!("[main] First run detected — skipping sidecar, showing setup wizard");
            } else {
                // Spawn sidecar in background
                let app_handle2 = app_handle.clone();
                tauri::async_runtime::spawn(async move {
                    match sidecar::spawn_sidecar(&app_handle2, env_vars).await {
                        Ok(child) => {
                            println!("[main] Sidecar spawned, waiting for health check...");
                            if let Err(e) = sidecar::wait_for_ready(30).await {
                                eprintln!("[main] Sidecar health check failed: {e}");
                                return;
                            }
                            // Store child handle for cleanup
                            let state = app_handle2.state::<SidecarState>();
                            *state.0.lock().unwrap() = Some(child);
                        }
                        Err(e) => eprintln!("[main] Failed to start sidecar: {e}"),
                    }
                });
            }

            Ok(())
        })
        .on_window_event(|window, event| {
            // Kill sidecar on app close
            if let tauri::WindowEvent::Destroyed = event {
                let state = window.state::<SidecarState>();
                if let Some(child) = state.0.lock().unwrap().take() {
                    let _ = child.kill();
                    println!("[main] Sidecar killed on window close");
                }
            }
        })
        .run(tauri::generate_context!())
        .expect("error while running tauri application");
}
```

```typescript
// Tauri bridge utilities — safe to import in browser (no-ops when not in Tauri)

let invoke: ((cmd: string, args?: Record<string, unknown>) => Promise<unknown>) | null = null

// Lazy-load Tauri API only when running inside Tauri
async function getTauriInvoke() {
  if (invoke) return invoke
  try {
    const mod = await import('@tauri-apps/api/core')
    invoke = mod.invoke
    return invoke
  } catch {
    // Not running in Tauri (e.g. dev browser)
    return null
  }
}

export async function sendNotification(title: string, body: string) {
  const inv = await getTauriInvoke()
  if (!inv) {
    // Fallback: browser Notification API
    if ('Notification' in window && Notification.permission === 'granted') {
      new Notification(title, { body })
    }
    return
  }
  try {
    // Use Tauri notification plugin
    const { sendNotification: tauriNotify } = await import('@tauri-apps/plugin-notification')
    await tauriNotify({ title, body })
  } catch (e) {
    console.warn('Tauri notification failed:', e)
  }
}

export async function isRunningInTauri(): Promise<boolean> {
  return (await getTauriInvoke()) !== null
}
```
