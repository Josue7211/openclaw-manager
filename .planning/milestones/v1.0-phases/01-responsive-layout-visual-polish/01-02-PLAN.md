**4. `Toast.tsx`** — Toast system with context provider:

```typescript
// Types
interface Toast {
  id: string
  type: 'success' | 'error' | 'warning' | 'info'
  message: string
  action?: { label: string; onClick: () => void }
  persistent?: boolean
}

type ToastPosition = 'top-left' | 'top-right' | 'bottom-left' | 'bottom-right'

// Context
const ToastContext = createContext<{ show: (toast: Omit<Toast, 'id'>) => void } | null>(null)

// Hook
export function useToast() {
  const ctx = useContext(ToastContext)
  if (!ctx) throw new Error('useToast must be used within ToastProvider')
  return ctx
}
```

ToastProvider behavior:
- Reads position from `localStorage.getItem('toast-position')`, defaults to `'top-left'`
- Only 1 toast visible at a time (replace mode: new toast replaces current)
- Auto-dismiss after 5 seconds unless `persistent: true`
- Toast anatomy: icon (left, from Phosphor if available, else Unicode symbols) + message text + optional action button + dismiss X button
- Toast styles: `background: 'var(--bg-card-solid)'`, `border: '1px solid var(--border)'`, `borderRadius: 'var(--radius-lg)'` (12px), `boxShadow: 'var(--shadow-high)'`, `zIndex: 'var(--z-toast)'` (5000)
- Icon colors: success = `var(--green)`, error = `var(--red-500)`, warning = `var(--amber)`, info = `var(--blue)`
- Position mapping: top-left = `{ top: 16, left: 16 }`, top-right = `{ top: 16, right: 16 }`, etc. as `position: fixed`
- Enter animation: use `animation: 'toastIn 0.3s var(--ease-spring) both'` (existing keyframe in globals.css)
- Dismiss animation: `animation: 'toastOut 0.2s ease forwards'` (existing keyframe)
- Accessibility: `role="status"` with `aria-live="polite"` for success/info, `role="alert"` with `aria-live="assertive"` for error/warning
- Dismiss X button: `<button>` with `aria-label="Dismiss notification"`

**5. `ProgressBar.tsx`** — Thin navigation progress bar:

Uses `useLocation()` from react-router-dom to detect route changes (NOT `useNavigation()` which is only for data routers).

```typescript
import { useEffect, useState, useRef } from 'react'
import { useLocation } from 'react-router-dom'

export function NavigationProgressBar() {
  const location = useLocation()
  const [visible, setVisible] = useState(false)
  const [progress, setProgress] = useState(0)
  const isFirstRender = useRef(true)

  useEffect(() => {
    // Skip first render (initial page load)
    if (isFirstRender.current) {
      isFirstRender.current = false
      return
    }

    setVisible(true)
    setProgress(30)
    const t1 = setTimeout(() => setProgress(60), 150)
    const t2 = setTimeout(() => setProgress(90), 350)
    const t3 = setTimeout(() => {
      setProgress(100)
      setTimeout(() => { setVisible(false); setProgress(0) }, 200)
    }, 500)

    return () => { clearTimeout(t1); clearTimeout(t2); clearTimeout(t3) }
  }, [location.pathname])

  if (!visible) return null

  // Render: position fixed, top 0, left 0, right 0, height 2px
  // Background: var(--accent), z-index: var(--z-max), pointer-events: none
  // Width transitions smoothly based on progress percentage
  // role="progressbar", aria-valuenow={progress}, aria-label="Loading page"
}
```

Height: 2px. Bar at very top of viewport, above everything.
