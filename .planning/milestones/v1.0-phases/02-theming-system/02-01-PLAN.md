1. Create `frontend/src/lib/theme-store.ts` following the exact `useSyncExternalStore` pattern from `keybindings.ts`:

   ```typescript
   import { useSyncExternalStore } from 'react'
   import type { ThemeState, ThemeDefinition, UserThemeOverrides } from './theme-definitions'

   const STORAGE_KEY = 'theme-state'
   ```

   **State management functions:**
   - `loadInitialState()`: Read from localStorage key `theme-state`, parse JSON, return ThemeState. Fallback: `{ mode: 'dark', activeThemeId: 'default-dark', overrides: {}, customThemes: [] }`
   - `persist()`: Write `_state` to localStorage, notify all listeners
   - `getThemeState()`: Return current `_state`
   - `subscribeTheme(fn)`: Add to `_listeners` set, return unsubscribe function
   - `setActiveTheme(id, clickEvent?)`: Update `activeThemeId`, set `lastModified: Date.now()`, persist. The `clickEvent` parameter `{ clientX: number; clientY: number }` is stored temporarily for the ripple animation (will be consumed by Plan 02's applyTheme)
   - `setMode(mode)`: Update `mode` field, set lastModified, persist
   - `setAccentOverride(color)`: Get or create override for `activeThemeId`, set `accent`, persist
   - `setGlowOverride(color)`: Same pattern for glow
   - `setSecondaryOverride(color)`: Same pattern for secondary
   - `setLogoOverride(color)`: Same pattern for logo
   - `setFontOverride(slot, fontFamily)`: Set font for a specific slot (body/heading/mono/ui) in override
   - `setFontScale(scale)`: Set fontScale (0.8-1.2) in override
   - `setGlobalFontOverride(enabled)`: Toggle globalFontOverride flag
   - `addCustomTheme(theme)`: Add to customThemes array
   - `removeCustomTheme(id)`: Remove from customThemes by id
   - `setPageOverride(routePath, themeId)`: Set pageOverrides[routePath] = themeId
   - `clearPageOverride(routePath)`: Delete pageOverrides[routePath]
   - `setCategoryOverride(categoryId, themeId)`: Set categoryOverrides[categoryId] = themeId
   - `clearCategoryOverride(categoryId)`: Delete
   - `setSchedule(schedule)`: Set schedule
   - `resetThemeOverrides(themeId)`: Delete overrides[themeId]
   - `pinTheme(themeId)`: Set overrides[themeId].pinned = true
   - `unpinTheme(themeId)`: Set overrides[themeId].pinned = false

   **React hook:**
   ```typescript
   export function useThemeState(): ThemeState {
     return useSyncExternalStore(subscribeTheme, getThemeState)
   }
   ```

   **Important:** Every mutation function must update `lastModified: Date.now()` on the state before persisting. This enables conflict resolution in preferences-sync (Pitfall #3 from RESEARCH.md).
