<objective>
Switch all 4 cron CRUD operations from HTTP REST proxy / CLI exec to gateway WebSocket RPC using the correct protocol v3 method names: cron.list, cron.add, cron.update, cron.remove.

Purpose: The current cron backend uses two wrong approaches: (1) GET /api/crons calls the `openclaw cron list --json` CLI binary via `tokio::process::Command`, and (2) POST/PATCH/DELETE use `gateway_forward()` which does HTTP REST calls to the workspace API, not WS RPC. The real gateway protocol v3 exposes cron operations as WS RPC methods (cron.list, cron.add, cron.update, cron.remove). This phase switches all four to use `state.gateway_ws.request()` like the session/agent handlers in gateway.rs.

Output: Rewritten crons.rs with all 4 handlers using gateway WS RPC, removed cron GET from openclaw_cli.rs.
</objective>

    ```rust
    /// `GET /api/crons`
    ///
    /// Lists all cron jobs via gateway WS RPC `cron.list`.
    /// Response: `{ "jobs": [...] }`
    async fn list_crons(
        State(state): State<AppState>,
        RequireAuth(_session): RequireAuth,
    ) -> Result<Json<Value>, AppError> {
        let gw = state.gateway_ws.as_ref().ok_or_else(|| {
            AppError::BadRequest(
                "OpenClaw Gateway not configured. Set OPENCLAW_WS in Settings > Connections.".into(),
            )
        })?;

        let payload = gw
            .request("cron.list", json!({}))
            .await
            .map_err(|e| {
                tracing::error!("[gateway] cron.list failed: {e}");
                AppError::BadRequest(format!("Gateway error: {}", sanitize_error_body(&e)))
            })?;

        // Gateway returns the list directly; wrap in { jobs: [...] } for frontend compatibility
        let jobs = if payload.is_array() {
            payload
        } else {
            payload.get("jobs").cloned()
                .or_else(|| payload.get("data").cloned())
                .unwrap_or_else(|| json!([]))
        };

        Ok(Json(json!({ "jobs": jobs })))
    }
    ```

    ) -> Result<Json<Value>, AppError> {
        if body.name.trim().is_empty() {

        let gw = state.gateway_ws.as_ref().ok_or_else(|| {
            AppError::BadRequest(

            )
        })?;

        let params = json!({

            "description": body.description,
            "schedule": body.schedule,
        });

        let payload = gw
            .request("cron.add", params)

                tracing::error!("[gateway] cron.add failed: {e}");
                AppError::BadRequest(format!("Gateway error: {}", sanitize_error_body(&e)))
            })?;

        Ok(Json(json!({ "ok": true, "job": payload })))
