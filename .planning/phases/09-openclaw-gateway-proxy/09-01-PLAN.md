**2. validate_gateway_path():**
```rust
pub(crate) fn validate_gateway_path(path: &str) -> Result<&str, AppError>
```
- Reject if contains `..` (path traversal)
- Reject if contains `?` or `#` (query/fragment injection)
- Reject if contains null byte `\0`
- Reject if contains `\n` or `\r` (CRLF injection)
- Must start with `/`
- On rejection: `AppError::BadRequest("invalid gateway path".into())`
- On success: return the path unchanged

**3. sanitize_error_body():**
```rust
pub(crate) fn sanitize_error_body(body: &str) -> String
```
- Step 1: Call `crate::redact::redact(body)` to handle API keys, JWTs, hex strings
- Step 2: Replace internal IPs using a static OnceLock<Regex> matching `\b(100\.\d{1,3}\.\d{1,3}\.\d{1,3}|10\.\d{1,3}\.\d{1,3}\.\d{1,3}|192\.168\.\d{1,3}\.\d{1,3})\b` with `[redacted-ip]`
- Step 3: Replace Unix file paths using a static OnceLock<Regex> matching `(?:\/[\w\-.]+){3,}` with `[path]`
- Step 4: Truncate stack traces -- if the result contains `"   at "` (3 spaces + "at "), take only the first line (split on `\n`, take first)
- Step 5: Truncate to 500 chars max (prevent response body flooding)
- Return the sanitized string

**4. gateway_forward():**
```rust
pub(crate) async fn gateway_forward(
    state: &AppState,
    method: Method,
    path: &str,
    body: Option<Value>,
) -> Result<Value, AppError>
```
- Call `validate_gateway_path(path)?` first
- Get base URL via `openclaw_api_url(state)` -- if None, return `AppError::BadRequest("OpenClaw API not configured. Set OPENCLAW_API_URL in Settings > Connections.".into())`
- Get API key via `openclaw_api_key(state)`
- Build URL: `format!("{base}{path}")` (base already has no trailing slash from `filter(!is_empty)`)
- Build request using `state.http.request(method, &url)` -- use `state.http` (bare reqwest), NOT `state.openclaw` ServiceClient (per research: retry on 5xx is dangerous for writes)
- Add `Content-Type: application/json` header
- If api_key is non-empty, add `Authorization: Bearer {api_key}` header
- If body is Some, call `.json(&body)` on the request builder
- Set timeout to 30 seconds: `.timeout(Duration::from_secs(30))`
- Send the request. On network error: `tracing::error!("[gateway] request to {path} failed: {e}")`, return `AppError::Internal(anyhow!("Failed to reach OpenClaw API"))`
- On non-success status (4xx or 5xx):
  - Read body text: `res.text().await.unwrap_or_default()`
  - Log: `tracing::error!("[gateway] {method} {path} -> {status}: {text}")`
  - Sanitize: `let safe_msg = sanitize_error_body(&text)`
  - For 4xx: return `AppError::BadRequest(format!("OpenClaw: {safe_msg}"))` (user-visible)
  - For 5xx: return `AppError::Internal(anyhow!("OpenClaw API error"))` (hidden from client)
- On success: parse JSON with `res.json::<Value>().await`, on parse error return `AppError::Internal`

**5. Health route:**
```rust
async fn openclaw_health(
    State(state): State<AppState>,
    RequireAuth(_session): RequireAuth,
) -> Result<Json<Value>, AppError>
```
- If `openclaw_api_url(&state)` is None: return `Ok(Json(json!({"ok": false, "status": "not_configured"})))`
- Otherwise: probe `{base}/health` with `state.http.get(url).timeout(Duration::from_secs(5)).send().await`
- On success (2xx): return `Ok(Json(json!({"ok": true, "status": "connected"})))`
- On any error or non-2xx: return `Ok(Json(json!({"ok": false, "status": "unreachable"})))`
- Health checks ALWAYS return HTTP 200 -- the "ok" field indicates status
