## Decisions Made
- var(--green-500) mapped to var(--secondary-dim) to preserve the "darker green" semantic
- var(--green-bright) mapped to var(--secondary-bright) for bright status text
- var(--accent-blue) mapped to var(--tertiary) as the primary tertiary color
- var(--blue-bright) mapped to var(--tertiary-bright) for bright accent text
- var(--accent-secondary) mapped to var(--tertiary) since it served the same role
- CSS .badge-green and .badge-blue switched from hardcoded rgba() to var(--secondary-a12) and var(--tertiary-a12)
