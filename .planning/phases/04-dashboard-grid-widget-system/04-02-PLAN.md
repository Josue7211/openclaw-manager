       - Function `getOrCreateLazy(pluginId: string)`:

         - Otherwise, look up WidgetDefinition via `getWidget(pluginId)`, create `React.lazy(def.component)`, cache it, return it

       - Export `WidgetWrapper` component (React.memo'd):
         ```typescript
         interface WidgetWrapperProps {
           widgetId: string     // Instance ID on the grid
           pluginId: string     // Widget type ID from registry
           config: Record<string, unknown>
           isEditMode: boolean
           size: { w: number; h: number }
         }
         ```

       - Implementation:
         - Call `getWidget(pluginId)` -- if undefined, return null
         - Call `getOrCreateLazy(pluginId)` to get LazyWidget
         - Render:
           ```tsx
           <div role="article" aria-label={def.name} style={{ width: '100%', height: '100%' }}>
             <PageErrorBoundary fallback={
               <ErrorState heading="Widget error" body="This widget couldn't load. Try removing and re-adding it." />
             }>
               <Suspense fallback={<WidgetSkeleton />}>
                 <LazyWidget widgetId={widgetId} config={config} isEditMode={isEditMode} size={size} />
               </Suspense>
             </PageErrorBoundary>
           </div>
           ```

       - `WidgetSkeleton` component: A simple div with `className="skeleton"` and `style={{ width: '100%', height: '100%', borderRadius: 'var(--radius-lg)' }}`
