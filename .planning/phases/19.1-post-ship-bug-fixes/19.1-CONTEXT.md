<decisions>
## Bug List

### Bug 1: "Executable not found: 'ffir'" error toast
- Persistent error notification on every page
- Something trying to run 'ffir' binary that doesn't exist
- Need to find the source and either fix the path or remove the call

### Bug 2: Chat page has redundant "Bjorn" tab
- Chat page shows "Chat" and "Bjorn" tabs
- The chat bot IS Bjorn — remove redundant tab, keep single Chat/Bjorn experience

### Bug 3: Dashboard shows only 1 widget in browser mode
- Tauri app shows all widgets, browser shows only Memory or Idea Briefing
- Likely a layout/localStorage issue for browser sessions

### Bug 4: OpenClaw Agents tab empty / loading forever
- Shows "AGENTS" + "LIVE PROCESSES loading..." but no agent cards
- Works in Tauri app but not browser — probably auth/session related

### Bug 5: OpenClaw Models/Usage/Tools say "not configured"
- "OpenClaw is not configured. Set OPENCLAW_API_URL in Settings > Connections"
- Health check may be checking wrong host or wrong env var
- Architecture note: OpenClaw services are on Services VM, NOT OpenClaw VM

### Bug 6: Remote Viewer architecture wrong
- Currently uses noVNC (VncViewer.tsx, useVnc hook)
- Should use Moonlight/Sunshine for remote desktop streaming to OpenClaw VM
- OpenClaw VM ONLY runs Moonlight/Sunshine — nothing else
- Need to replace noVNC with Moonlight/Sunshine web client or embed approach

### Bug 7: Skills/Marketplace/Plugins don't exist
- No pages for skills marketplace, plugin install, or plugin management
- If referenced in navigation/sidebar, remove broken references
- If actual pages exist but are broken, fix them

### Bug 8: Models page is read-only cards
- Even when "working", ModelsTab only shows static model cards
- Should show real model data from LiteLLM API when connected
- Needs to actually pull model list from the gateway

### Bug 9: Duplicate Agents page
- Agents exists as standalone page AND embedded in OpenClaw tabs
- Deduplicate — single source of truth for agent management
- OpenClaw tab should be the primary location

### Bug 10: koel.rs untracked
- New music streaming route exists but not committed
- Either commit it (if ready) or gitignore it (if WIP)

### Bug 11: Architecture correction needed
- CLAUDE.md says OpenClaw VM runs "AI agents, OpenClaw, LiteLLM"
- REALITY: OpenClaw VM ONLY runs Moonlight and Sunshine
- All services (LiteLLM, OpenClaw gateway) are on Services VM
- App config and health checks need to point at correct hosts

### Claude's Discretion
All implementation choices — these are bug fixes, not feature decisions.

</decisions>
