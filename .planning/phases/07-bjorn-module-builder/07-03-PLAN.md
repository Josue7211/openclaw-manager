  <action>
    1. Create `frontend/src/pages/chat/BjornPreview.tsx`:

       Props interface:
       ```typescript
       interface BjornPreviewProps {
         source: string              // Generated component source code
         generationState: BjornGenerationState
       }
       ```

       Component implementation:
       - Use `useRef<HTMLIFrameElement>(null)` for iframe reference
       - Use `useMemo` to run `analyzeCode(source)` when source changes -- memoize the AnalysisResult
       - Use `useMemo` to build srcdoc HTML via `buildSandboxHTML(source, getThemeVarsCSS())` only when analysis.safe is true
       - Use `useEffect` to set up postMessage listener:
         a. On 'message' event, check `event.source === iframeRef.current?.contentWindow` (source validation instead of origin -- srcdoc has opaque "null" origin)
         b. If `event.data?.type === 'data-request'`, extract requestId, source, command
         c. Call `api.post('/api/bjorn/bridge', { source: event.data.source, command: event.data.command })`
         d. On success: `iframeRef.current?.contentWindow?.postMessage({ type: 'data-response', requestId, data: result }, '*')`
         e. On error: `iframeRef.current?.contentWindow?.postMessage({ type: 'data-error', requestId, error: err.message }, '*')`
         f. Clean up listener on unmount

       Render logic:
       - If `generationState === 'generating'`: Show skeleton loading state with pulsing animation and "Generating module..." text
       - If `!source`: Show empty state with Robot icon and "Describe a module to see a preview" text
       - If `!analysis.safe`: Show violations panel -- a scrollable list showing each violation's line number and snippet in a code-style display, with a red warning banner at top ("Static analysis found N issue(s)")
       - If `analysis.safe`: Render `<iframe ref={iframeRef} sandbox="allow-scripts" srcDoc={sandboxHtml} style={{ width: '100%', height: '100%', border: 'none', borderRadius: '8px', background: 'var(--bg-card)' }} title="Module Preview" />`

       Style the container with: flex: 1, minWidth: 0, display: flex, flexDirection: column, background: var(--bg-panel), borderRadius: 12px, overflow: hidden. Add a header bar with "Preview" label.
