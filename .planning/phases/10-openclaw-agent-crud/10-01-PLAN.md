Add three new handlers to agents.rs and wire them into the router:

1. **`validate_agent_id()`** helper function (private):
   - Accepts alphanumeric chars, hyphens, and underscores, 1-64 chars
   - Regex: `^[a-zA-Z0-9_-]{1,64}$`
   - Returns `Result<&str, AppError>` with `AppError::BadRequest("invalid agent id")`
   - Use `OnceLock<regex::Regex>` like validate_uuid does
   - Reason: existing seed agents use IDs like 'koda', 'fast' -- validate_uuid would reject them

2. **`create_agent` handler** (POST /agents):
   - Deserialize `CreateAgentBody { display_name: String, emoji: Option<String>, role: Option<String>, model: Option<String> }`
   - Generate UUID v4 for the agent ID via `uuid::Uuid::new_v4().to_string()`
   - Generate a lowercase `name` from display_name: take first word, lowercase, truncate to 20 chars, replace non-alphanumeric with underscore. If empty, use "agent"
   - Get `sort_order` by counting existing agents for this user + 1: `SELECT COALESCE(MAX(sort_order), 0) + 1 FROM agents WHERE user_id = ? AND deleted_at IS NULL`
   - INSERT into agents table with user_id from session, status='idle', current_task='', created_at/updated_at = now (chrono::Utc::now().to_rfc3339())
   - Call `crate::sync::log_mutation(&state.db, "agents", &id, "INSERT", Some(&payload))` with the full agent JSON as payload
   - Return `Json(json!({ "agent": { ...all fields... } }))` with 200 status

3. **`delete_agent` handler** (DELETE /agents):
   - Deserialize `DeleteAgentBody { id: String }`
   - Validate with `validate_agent_id(&body.id)?` (NOT validate_uuid)
   - Soft delete: `UPDATE agents SET deleted_at = ?, updated_at = ? WHERE id = ? AND user_id = ? AND deleted_at IS NULL`
   - Check rows_affected -- if 0, return `AppError::BadRequest("agent not found")`
   - Call `crate::sync::log_mutation(&state.db, "agents", &body.id, "DELETE", None)`
   - Return `Json(json!({ "ok": true }))`

4. **`agent_action` handler** (POST /agents/action):
   - Deserialize `AgentActionBody { id: String, action: String }`
   - Validate with `validate_agent_id(&body.id)?`
   - Validate action is one of: "start", "stop", "restart" -- else return BadRequest
   - Call `gateway_forward(&state, Method::POST, &format!("/agents/{}/action", body.id), Some(json!({"action": body.action})))`
   - Import `reqwest::Method` and `super::gateway::gateway_forward`
   - Return the gateway response as-is: `Ok(Json(result))`
